Coin-Kept is a personal finance web application that helps users
track income and expenses and manage their financial records easily.

This repository contains the frontend of the Coin-Kept application.

## Tech Stack
- React
- JavaScript
- Tailwind CSS
- Vite
- Axios

## Features
- User authentication (login / register)
- Add, edit, and delete income & expense records
- View transaction history
- Summary of income and expenses
- Responsive user interface

## Project Overview
- Built reusable UI components using React
- Implemented forms for financial data input
- Integrated frontend with backend REST APIs
- Managed application state and error handling
